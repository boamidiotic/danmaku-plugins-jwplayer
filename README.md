# Danmaku Plugins For JW Player
[![Secreenshot](https://i.imgur.com/v2y5Ind.png)](https://fb.me/xnongermx1337/)
### Usage
```html
<div id="boamganteng"></div>
<script>
jwplayer("boamganteng").setup({
    file: "/uploads/example.mp4",
    image: "/uploads/example.jpg",
    plugins: {
        "/plugins/danmaku.js": {
           danmaku: true,
        }
    }
});
</script>
```
